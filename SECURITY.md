# Security Policy

## Supported Versions

The latest production deployment at https://gate-tracker-e1a99.web.app is the only supported version.

## Reporting a Vulnerability

Please report security vulnerabilities by emailing:

**security@gate-tracker-e1a99.web.app**

You can also use the contact information in:
- `/.well-known/security.txt` (on the live site)
- `public/.well-known/security.txt` (in this repository)

### What to include
- Description of the vulnerability
- Steps to reproduce (theoretical, do not test on production)
- Affected component or file
- Proposed fix (if known)

### Response timeline
- Acknowledgment within 72 hours
- Status update within 5 business days
- Fix timeline communicated based on severity

## Scope

This security policy covers the GATE CSE 2027 Preparation Tracker application,
including all code in this repository, Firebase services configuration,
and the production deployment.

## Out of scope
- Social engineering attacks against users
- Physical attacks against infrastructure
- Third-party services (Google Firebase platform itself)
